Android SDK 镜像更新

当遇到代理不稳定的情况，可以采用国内镜像资源更新Android SDK。不得不说，有时候镜像资源的速度还是非常给力的。这里推荐一下：国内镜像资源。

Step 1

打开Android SDK Manager 选择 Tools –> Options,然后在 Proxy Settings的Http Proxy Server和Http Proxy Port分别填上mirrors.neusoft.edu.cn和80，勾选上 [Force https://… sources to be fetched using http]，然后点击Close。

Step 2

然后选择Android SDK Manager下的Packages和Reload，这时候就可以更新了。
